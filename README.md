# 🎯 Customer Churn Analytics Dashboard
✨ Welcome to my **E-Commerce Customer Churn Prevention** project, where customer behavior data transforms into retention strategy. This project demonstrates how Power BI can unlock the hidden patterns behind customer loss 📊.

## ✨ Project Overview
This Power BI dashboard analyzes **6,000 e-commerce customers**, offering a clear lens into:
- Which customer segments are most at-risk of leaving
- What factors predict churn (satisfaction, engagement, support issues)
- How much revenue is at stake with each segment
- The impact of loyalty programs on retention
- Which behavioral signals indicate imminent churn

## 🔍 Key Insights Uncovered
- **Overall Churn Rate:** 15.48% of customers have churned
- **Satisfaction Impact:**
  Dissatisfied customers (score < 4): 46.7% churn rate
  Very Satisfied customers (score 8-10): 12.0% churn rate
  **Difference: 35 percentage points!**
- **Engagement Crisis:**
  Low engagement customers: 63% churn rate
  Medium engagement: 1% churn rate
  **Engagement is the strongest churn predictor**
- **Support Tickets Signal Problems:**
  Churned customers averaged 1.11 support tickets
  Active customers averaged 0.81 support tickets
  **More tickets = more friction = higher churn**
- **Loyalty Program Works:**
  Non-members: 16.9% churn rate
  Loyalty members: 9.0% churn rate
  **7.9 percentage point reduction with loyalty**
- **Revenue at Risk:**
  $13,728 from At-Risk & Critical segments
  $3.5M from Healthy segment (largest risk if they churn)

## 📊 Dashboard Features
**Two Interactive Tabs:**

**Tab 1: Customer Segments**
- Customer distribution across segments (Healthy, At-Risk, Critical, Churned)
- Revenue by segment (identify highest-value losses)
- Satisfaction trends (quality of each segment)
- Activity levels (inactivity = churn risk)

**Tab 2: Churn Drivers**
- Satisfaction vs. Churn correlation (proves quality matters)
- Engagement impact (shows engagement critical for retention)
- Support ticket analysis (identifies frustrated customers)
- Loyalty program effectiveness (ROI of loyalty investment)

**Interactive Filters:**
- Tenure (customer age)
- Loyalty Program Status (members vs. non-members)
- Customer Value (high-value vs. regular customers)
- All filters apply across both tabs for deep analysis

## 📂 Why This Project Matters
Customer churn isn't just about lost sales — it's about lost lifetime value.

This project shows how a well-designed Power BI dashboard can:
- **Identify at-risk customers BEFORE they leave** (enabling proactive retention)
- **Quantify the cost of churn** (revenue at stake)
- **Pinpoint root causes** (satisfaction, engagement, support quality)
- **Measure intervention success** (loyalty program impact)
- **Drive strategic decisions** (where to invest retention budget)
